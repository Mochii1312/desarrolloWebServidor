Pregunta 1 (5 puntos): Configuración de Entorno de Desarrollo
Paso 1: Descargar Xampp desde su pagina oficial
Paso 2: Una vez la descarga este completa ejecuta el instalador como administrador, esto iniciara la instalacion de Xampp
Paso 3: Selecciona los componentes
-Apache: Servidor web que ejecutara PHP
-MySql: Sistema de gestion de base de datos
PHP: Lenguaje principal de programacion
phpMyAdmin: Interfaz web para la gestion de MySql
FileZilla FTP Server: Para transferencias de archivos(No obligatorio)
Paso 4: Selecciona la ubicacion donde instalar Xampp, la ubicacion predeterminada (C:\xampp) es la mas recomendada
Paso5: Verifica que los puertos 80(Apache) y 3306(Mysql) esten disponibles. Si hay problemas configura puertos alternativos
Paso 6:Inicio de los servicios: Desde el panel de control de Xampp Inicia Apache pulsando al boton de start, inicia MySql tambien pulsando al boton de start y verifica que ambos servicios muestren running y estan en verde
Paso 7: Verificacion e instalccion: Abre el navegador y navega a http://localhost y se deberia mostrar la pagina de bienvenida de Xampp.

paso 8: Estructura recomendada para el proyecto seria: C:\xampp\php\php.ini

Pregunta 6 (4 puntos): Configuración Apache y .htaccess
Archivo de texto de configuración de Apache, sirve para controlar el comportamiento de un directorio y subdirectorios.
La configuración básica de reescritura de URL permite transformar URLs dinámicas en amigables para el usuario.
La configuración de seguridad básica evita que se vea el contenido de carpetas, protege archivos importantes, restringe acceso por IP, evita que se ejecuten scripts en zonas seguras.
Apache tiene configuración .htaccess, usa modulos mod_php, mod_rewrite.
Nginx tiene mayir rendimiento y configuración centralizada.
Comandos terminal: 
-sudo systemctl start apache2 → inicia Apache
-sudo systemctl stop apache2 → lo detiene
-sudo systemctl restart apache2 → lo reinicia
-sudo systemctl status apache2 → muestra su estado
